🛒 E-commerce Backend API

This is a backend system for an E-commerce application built with a focus on scalability and clean architecture. The platform supports two main user panels:

Buyer Panel
Seller Panel
🚀 Features
👤 Buyer Panel
Browse products
View product details
Purchase products

---

🛍️ Seller Panel

Sellers have full control over product management:

➕ Add single product
📦 Add multiple products (Bulk upload)
🗂️ Assign products to categories
✏️ Update product details
❌ Delete products
💰 Manage product sales

---

📦 Product Structure

Each product contains the following fields:

Product Name
Product Image
Product Price
Product Rating
Product Description
Product Discount
Category ID

---

🛠️ Core Functionalities

RESTful APIs for all operations
CRUD operations for products
Category-based product management
Bulk product handling
Clean and scalable backend structure
